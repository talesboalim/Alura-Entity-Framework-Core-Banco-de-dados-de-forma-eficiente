# Curso Alura - Entity Framework Core: Banco de dados de forma eficiente

## Requisitos
Visual Studio 2019
Microsoft SQL Server 2019

## Instalação
-Restaurar o LojaDB.BAK do arquivo ./LojaDB.zip no MSSQL 2019
-Abrir projeto ./Alura.Loja/Alura.Loja.sln
-Configurar conexão com o banco MSSQL em LojaContext.cs
-Instalar pacotes Nuget referenciados no projeto
